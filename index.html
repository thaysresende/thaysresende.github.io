Primeira Dashboard - SP Trans

Tecnologias utilizadas:

Primeira dashboard construída utilizando algumas funções da framework Vue.js, consumo de API Rest e utilização da Maps JavaScript API da Google. A API Olho Vivo foi consumida utilizando a tecnologia fetch. O objetivo da dashboard é fornecer uma plataforma para visualização da localização dos ônibus da cidade de São Paulo, previsão de chegada e pontos de parada das linhas.

Projeto de Conclusão de Curso

Tecnologias utilizadas:

Projeto de conclusão de curso realizado sob a supervisão do Professor Me. Rodrigo Porfírio da Silva Sacchi. Foi desenvolvida uma Explorable Explanation sobre tópicos iniciais da disciplina de sistemas distribuídos. Uma explorable explanation é um web app interativo que reúne conteúdos descritivos e pequenas simulações interativas que permitem que o usuário teste e observe o conteúdo do site. O web app foi desenvolvido também por Gabriele Paloma Vieira Rodrigues.
